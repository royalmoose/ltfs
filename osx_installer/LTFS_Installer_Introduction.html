This is the installation package for the
Quantum LTFS software version 2.4.0.2

This will install the Quantum LTFS software onto your system. Note that to use Quantum LTFS software, you should also have already installed the following packages:

  • OSXFUSE, version 3.6.3.
  • ICU version 4.8.1.1 (attached to this distribution package).

This installer will not allow you to proceed until you have installed these two packages.
